# A pytorch implementation of the GrassNet Algorithm 

## Dependencies

- PyTorch 1.0+
- tqdm
- DGL
- networkx
- mamba_ssm (https://github.com/state-spaces/mamba)

```bash
pip install torch
pip install tqdm
pip install dgl
pip install networkx
pip install mamba-ssm
```

---


## Node Classification

```bash
python main.py --dataset cora 
python main.py --dataset citeseer 
```
